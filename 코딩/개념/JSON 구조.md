#JSON구조
JSON은 **데이터를 표현하는 텍스트 형식**으로, 두 가지 기본 요소로 이루어져 있습니다

1. **객체(Object)**
    - 중괄호 `{}`로 감싸서 표현
    - 내부는 `"키": 값` 쌍으로 구성
```json
{
      "name": "홍길동",
      "age": 25,
      "isStudent": true
}
 ```
2. **배열(Array)**
    - 대괄호 `[]`로 감싸서 표현
    - 값들을 순서대로 나열
```json
[
    "사과",
    "바나나",
    "포도"
]
```
