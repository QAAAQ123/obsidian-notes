프론트엔드와 클라이언트는 종종 같은 의미로 사용되지만, 정확히는 다른 개념입니다.

- **클라이언트(Client)**는 서버에 서비스를 요청하는 모든 주체를 의미합니다. 웹 브라우저, 모바일 앱, 데스크톱 애플리케이션 등이 클라이언트가 될 수 있습니다. 즉, 하드웨어 또는 소프트웨어의 역할에 가까운 넓은 개념입니다.
    
- **프론트엔드(Frontend)**는 클라이언트의 한 부분으로, 사용자에게 직접 보여지는 화면과 사용자와 상호작용하는 부분을 개발하는 영역을 말합니다. 이는 웹 개발에서 주로 사용되는 용어이며, HTML, CSS, JavaScript 등으로 구현됩니다.
    

쉽게 말해, 웹 브라우저가 **클라이언트** 역할을 하고, 웹 브라우저에 보이는 화면을 구성하는 기술이 **프론트엔드**입니다.
