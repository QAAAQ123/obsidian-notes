## Concept of Sets in redis
- unorder collection of unique Strings
- no duplication allow
![[Pasted image 20260226143816.png]]

### 명령어
### sadd:Set add
`sadd {redis key} {member1} {member2}....`

### smembers: Set memebers
`smembers {reids key}`: set key에 해당하는 value들 전체 확인

## scard
`scard {redis key}`: key에 몇개의 value가 있는지 확인
- return값: integer

### sdiff: Set difference
`sdiff {Object redis key} {subject redis key1} {subject redis key2}...`:다른 set과 비교해서 Object reids key**에만** 있는 value들을 보여줌

#### sdiffstore: Set differce store
`sdiffstore {New redis key} {existed object redis key} {existed subject redis key1} {existed subject redis key2} ..`: 새로운 key에 sdiff한 결과 저장함

### sunion: Set union
`sunion {redis key1} {reids key2} {redis key3} ...`: key에 해당하는 모든 value들을 합침

#### sunionstore: Set union store
`sunionstore {New redis key} {existed redis key1} {existed redis key2} {existed redis key3} ..`: 새로운 key에 sunion한 결과 저장함

### srem: Set remove
`srem {redis key} {member1} {member2} ....`

### spop: Set pop
`spop {redis key} {count}`: count 개수만큼 랜덤value를 set에서 pop한다

### sinter: Set intersection
`sinter {redis key1} {reids key2} {redis key3} ...`: 공통인 value들만 보여줌(교집합)
#### sinterstore

### smove: Set move from A to B
`smove {source key} {destination key} {member}`: A에 있는 값을 B로 옮김
