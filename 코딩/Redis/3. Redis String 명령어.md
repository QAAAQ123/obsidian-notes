#redis
- 대부분의 Redis Server는 Linux에 설치함
- Windows,Mac에서 명령어는 똑같음

### 시작
- Redis server 시작:redis-server
- Redis CLI 시작: redis-cli

### 명령어
1. keys {pattern}: key값들 찾기
	1. `*`: 와일드 카드 => `keys *`: 모든 key값 찾기
2. set {key} {value}: key에 String 타입의 value 저장
	1. set name3 tom: key는 name3 value는 String타입의tom으로 저장
3. get {key}: {key}에 해당하는 value 
4. del {key}: {key}에 해당하는 key-value 삭제
	1. 결과값은 몇개가 지워졌는지 보여줌
	2. `del key1 key2`형태로 여러개의 key-value 삭제 가능
5. flushall: 모든 key-value 삭제

### 주의 사항
- **set으로 이미 있는 key값을 쓰면 덮어써진다.**
- ex)set name tom한 후에 set name mark함
	- get name하면 tom이 아닌 mark가 나옴
