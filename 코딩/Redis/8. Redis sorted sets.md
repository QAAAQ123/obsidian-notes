## Concept of Sorted Sets in redis
![[Pasted image 20260226150545.png]]
- smiler to the redis sets with a unqiune feature that members can be sotred
- 모든 sorted set의 value는 score와 관련있다
- score 기반으로 정렬된다
- redis sorted set은 score와 value로 이루어져있다

### sorted set score
- sorted set에는 동일한 score의 member가 들어갈 수 있다. 하지만 인덱스는 다르다(zrank에서 index가 다르게 나온다)
ex) {score:1,member: aaa},{score:1,member: bbb} 가능하다

## 명령어: sorted set의 명령어는 `z`로 시작한다
### zadd
`zadd {redis key} {score1} {member1} {score2} {score2} ..`

### zrange
`zrange {redis key} {index} {index}`
zrange key 0 -1: 전체 보기

### zcard
`zcard {redis key}`: key에 몇개의 value있는지 보기

### zcount: score 기준으로 확인
`zcount {reids key} {start} {end}`:start부터 end에 해당하는 value들이 몇개 있는지 확인

### zrem
`zrem {redis key} {member1} {member2}...`

### zrank
`zrank {redis key} {member}`: 해당 member가 몇번째에 있는지 **index**를 나타냄

### zrevrank: reverse rank
`zrevrank {redis key} {member}`: zrank와 반대로 indexing

### zscore
`zscore {redis key} {member}`:member의 score가 뭔지 확인

### zrangebyscore 
`zrangebyscore {redis key} {min score} {max score}`: score범위내에서 정렬하여 보여줌
