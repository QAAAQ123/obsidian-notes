#redis
### Redis
1. 오픈소스
2. BSD 라이센스 
3. Key-Value 저장소
4. in-memory key-value 저장소
5. 영속성 지원
6. String,Hash,List,Set,SortedSet 등 지원
7. C로 작성됨
8. NoSQL

## NoSQL
- An informal,loosely-defined term for non-relational, structured data storage system
- MongoDB,memecached,CouchDB,**Redis**


## Redis mean?
- Remote Dictionary Server=>RE DI S
- key-value구조는 string,hash,list,set,sorted set을 담을 수 있음

## Redis used for?
- advanced key-value store
	- NoSQL
	- memory-cache store(시스템 메모리안에 저장된 데이터를 제공할 때 성능 향상을 위해)
	- message broker
## how to interact
- Install Redis Server
- Run Redis CLI
