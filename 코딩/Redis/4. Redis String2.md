#redis 
## setex: setting expiry key-value
- set {key} {expiry seconds} {value}
	- set name 10 max: 10초 후에 만료되는 name-max

## ttl: time to leave
- ttl {key}: 남은 시간(사라지면 마이너스임)
	- set name 10 max -> 12초후 -> ttl name의 결과는 -2

## setnx: set if not exists
- setnx {key}: 이미 해당하는 key가 만들어져있으면 덮어쓰거나 생성하지 않음
	- set name2 max -> setnx name2 john하면 결과는 OK가 아닌 0이 나옴

## strlen: string length
- strlen {key}: key에 해당하는 value의 char길이

## mset: Multiple SET
- mset {key1} {value1} {key2} {value2} and so on...

## psetex: p-set-ex
- psetex {key} {miliseconds} {value}:meiliseconds로 expiry key-value생성

## decr,incr
- decr {key}: value의 integer값을 1감소
- incr {key}: value의 interge값을 1증가

## incrby,decrby
- incrby {key} {integer}: value의 값을 interger만큼 증가
- decrby {key} {integer}: value의 값을 integer만큼 감소

## append
- append {key} {string}: 이미 있는 key-value에 string을 붙힘
	- set mykey hello -> append mykey " world" -> get mykey는 "hello world"
