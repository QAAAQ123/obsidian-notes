## 응답 바디 처리
- **모델에 반환할 데이터를 json형태로 만들어두고**
- **해당 ~={red}데코레이터 함수에 `response_model`로 넣는다=~**
```python
@app.get("/",response_model=Items)
#and so on
```

## 오류 처리
- **~={red}HTTPException 클래스를 사용=~**해 예외 처리한다
- HttpException 인수
	1. status_code
	2. detail: 클라이언트로 전송할 메시지
	3. headers
- **~={red}python의 raise로 처리(raise: 고의적으로 예외 발생 시키는 키워드)=~**
- `raise HTTPException(.........)`
