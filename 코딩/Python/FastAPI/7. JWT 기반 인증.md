### 인증과 허가(권한)
인증: 개체가 전달한 인증 정보를 검증
허가: 개체가 특정 처리를 할 수 있도록 권한 부여


## FastAPI의 인증 방식
1. 기본 HTTP 인증:사용자 인증 정보를 Authorization HTTP 헤더를 사용해 전송하는 방식
2. 쿠기: 데이터를 클라이언트 측에 저장할때 사용
3. bearer 토큰 인증: bearer토큰을 이용해 인증하는 방식,Authorization 헤더에 Bearer 키워드로 전송됨

## FastAPI의 의존성 주입
- 객체(함수)가 실행에 필요한 인스턴스 변수를 받는 방식
- 라우트 처리 함수의 인수로 의존 라이브러리를 주입
- 의존 라이브러리는 함수 또는 클래스로 정의됨
- fastapi의 Depends를 import하고 사용하면 됨 `async def get_user_detils(user: User = Depends(get_user)):

## CORS
- 교차 출처 리소스 공유: CORS
- 등록되지 않은 사용자가 리소스를 사용하지 못하도록 제한하는 규칙
- FastAPI는 CORSMiddleware CORS미들웨어를 사용하여 API에 접근가능한 출처 관리
- 미들웨어: 하나의 함수로, 특정 처리 사이의 중개자 역할을 한다. 웹 API에서 미들웨어는 요청과 응답간 중개자이다
