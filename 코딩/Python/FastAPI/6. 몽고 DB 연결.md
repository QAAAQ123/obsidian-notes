### beanie
- beanie는 ODM이다
- **ODM**(Object Document Mapper):객체 문서 매퍼


- 여기서는 beanie사용
- NoSQL은 데이터 저장을 위해 문서를 사용한다. 
- 문서는 pydantic모델과 동일한 방식으로 정의된다. 
- 차이점은 pydantic가 아닌 beanie의 Document를 사용한다는 것이다
```python
from beanie import Document
class Event(Document):
	name: str
	location: str
	
	class Settings:
		name="events"
```

### CRUD 메서드
1. insert(),create()
	1. insert_one(): 단일 데이터
	2. insert_many(): 여러개의 데이터
2. find(),get()
	1. find(): 인수로 저장한 문서를 목록에서 찾음
	2. get(): 지정한 ID와 일치하는 단일 문서 반환
	3. find_one(): 지정 조건 단일 문서 반환
3. save(),update(),upsert()
	1. save(): 신규 데이터 저장
	2. update(): 기존 문서 변경
	3. upsert(): 조건에 부합하는 문서가 없으면 신규 추가
4. delete(): 문서 삭제
