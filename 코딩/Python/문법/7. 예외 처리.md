```python
try:
	실행할 명령
except 예외 as 변수:
	오류 처리문
else: 
	예외가 발생하지 않을 때 처리
	

# 예시
str = "89"
try:
	score = int(str)
	print(score)
	a = str[5]
except ValueError:
	print("점수의 형식이 잘못되었습니다.")
except IndexError as e:
	print("첨자 범위를 벗어났습니다.\n에러 내용:{}", e)
print("작업 완료")
```

## raise
고의적으로 예외 발생시킨다
ex) if(n<0): raise ValueError

## finally
- 예외 발생과 상관없이 반드시 실행해야할 명령 지정
- 파일은 finally 대신에 with 주로 사용
```python
try:
	명령
finally:
	명령
```

## assert
- 프로그램의 현재 상태가 맞는지 확인
- ~={red}이 조건이 맞는지 지금 당장 확인하라는 뜻이다. 의심이 가는 곳에 assert를 작성하면 문제를 조기에 발견하는데 큰 도움이 된다.=~
- assert를 사용하면 실행이 느려진다. -O 옵션을 주면 assert문장을 무시한다.
- 디버깅할 때 확인용으로 사용하고 최종적으로 빼는 것이 좋다.
- **정상이면 아무 일도 하지 않지만, 거짓이면 AssertionError 예외를 발생시키고 즉시 중지한다.**
- `assert 조건,메시지`
```python
score = 128
assert score <= 100, "점수는 100이하여야 합니다."
print(score)
```
