## 사전(Dictionary) 구조
- Map
- 파이썬에서 기본 기능으로 제공해준다,**키는 읽기 전용이고 값은 자유롭게 변경이 가능**하다
-~={red} **{}를 사용하여 정의**=~한다.
- `dic = {'boy':'소년','school':'학교','book':'책'}` 

### 사전에서 값을 읽는 방법
1. `dic[값]`으로 읽기: 키가 없으면 에러 발생
2. ` dic.get(키)`로 읽기: 키가 없어도 에러가 발생하지 않음
3. `dic.keys()`로 모든 키 값 읽기
4. `dic.values()`로 모든 값 읽기
5. `dic.items()`로 모든 키:값 읽기

### 사전 특징
1. 변경 가능 자료형,but 키는 변경 불가
2. update()메소드로 두 개의 사전 병합 가능: dic.update(dic2)
