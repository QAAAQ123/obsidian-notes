- python의 리스트는 같은 리스트 안에 다른 타입을 담을 수 있다. ["이름",111,"나이",444]
- [begin:end:step]로 원하는 범위만 읽을 수 있다

#### 리스트 컴프리헨션
- 규칙을 가지는 수열이면 [수식 for 변수 in 리스트 if 조건]으로 리스트를 만들 수 있다
ex) [a for b in range(1,11)] => range(1,11)범위를 순회하며 b를 추출하여 a를 요소로 생성한다.
`[n*2 for n in range(1,11)]`: 1부터 10까지 순회하며 2배가 된느 값만 취하는 경우

## 튜플
- 리스트와 유사하지만 **~={red}초기화 후 편집 할 수 없는 구조=~**
- 리스트는 []로 감싸고,튜플은 ()로 감싼다
- 추출,일부 잘라내기,튜플끼리 연결,`*`로 요소 반복 가능
- **~={red}튜플의 요소를 변경하거나 삭제하는것은 불가능=~**

### 튜플을 사용하는 이유
1. 튜플은 내부 구조가 단순하고 읽는 속도가 빠르다
2. 편집할 수 없어 안정적이다. DB에서 읽은 정보나 네트워크로 받은 정보는 튜플로 받는 것이 안정적이다.
