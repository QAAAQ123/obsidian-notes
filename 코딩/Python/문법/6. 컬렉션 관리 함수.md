## enumerate
- enumerate: ~={yellow}**순서값과 요소값 둘을 한꺼번에 구해주는 내장 함수**=~
```python
race = ['저그','테란','프로토스']
list(enumerate(race))
```
=> 결과: `[(0,'저그'),(1,'테란'),(2,'프로토스')]`

## zip
- zip: 여러 개의 컬렉션을 합쳐서 하나로 만드는 함수
- 대응되는 요소끼리 짝을 지어 튜플의 리스트를 생성
- 짧은 쪽의 길이에 맞춰진다
```python
days = ["월","화","수","목","금","토","일"]
food = ["갈비탕","순대국","칼국수","삼겹살"]
menu = zip(days,food)
for y, f in menu:
	print("%s요일 메뉴: %s %(y,f))
```
=> 결과: 월요일 메뉴: 갈비탕\n화요일 메뉴: 순대국\n....

### filter
- filter: 리스트의 요소중 조건에 맞는 것만 골라냄
- filter(조건,리스트): 여기서 조건은 함수이다
```python
def flunk(s):
	return s < 60
	
score = [45,89,72,53,94]
for s in filter(flunk,score):
	print(s)
```
=> 결과; 45\n53

### map
- map: 모든 요소에 대해 변환 함수의 기능을 수행하고 결과 값으로 구성된 리스트 생성
- filter 함수와 사용법 동일(요소값 변경 함수를 만들고 리스트에 사용)
```python
def half(s):
	return s / 2
	
score = [45,89,72,53,94]
for s in map(half,score):
	print(s,end=', ')
```
결과=> 22.5, 44.5, 36.0, 26.5, 47.0

## ★★람다 함수★★
- 람다 함수: 이름이 없고 **입력과 출력만으로 함수를 정의하는 방법**
- `lambda 인수:식`
filter와 map의 함수를 람다 함수로 변경
```python
score = [45,89,72,53,94]
for s in filter(lambda x:x < 60,score):
	print(s)
	
for s in map(lambda x:x/2,score):
	print(s)
```


## shallow copy and deep copy in python
- 파이썬의 기본 타입은 깊은 복사를 한다
- 그러나 컬렉션은 얕은 복사를 하기 때문에 깊은 복사를 하기 위해서는 별도의 방법으로 복사해야 한다
	 1. collection.copy(): copy()메서드 이용
	 2. collection2 = collection1[:]: `[:]`이용
	 3. 가끔 복사해야할 리스트 안에 리스트가 있는 경우 copy.deepcopy()를 써야 한다.
	    list1=[list0,1,2] \n list2 = copy.deepcopy(list1) -> 이 경우 list2는 list0,1과 완전히 독립적이다
