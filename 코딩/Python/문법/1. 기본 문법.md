## 출력
`print(출력 내용,[sep=구분자],[end = 끝 문자])`
sep: 출력 내용 사이에 출력할 값 지정
end: 개행(\n)대신에 출력할 값 지정
```python
print('강아지','고양이',sep=' 그리고 ',end='이다')
```
출력 결과: 강아지 그리고 고양이이다.

## 입력
`변수 = input('내용')`을 사용한다.

```python
price = input('가격을 입력하세요')
num = input('개수를 입력하세요`)
sum = int(price) * int(num)
print('총액은',sum,'원입니다.')

#or 

price = int(input('가격을 입력하세요'))
num = int(input('개수를 입력하세요'))
sum = price * num
print('총액은',sum,'원입니다.')
```

## 변수
- 자동으로 타입을 찾아서 사용함
- 변수의 타입이 계속 변할 수 있음
- del '변수'로 메모리에서 변수 삭제 가능

## 타입
### 수치형
1. 정수형
2. 실수형
	1. 큰 수는 부동 소수점 방식 사용
	   정수값이여도 부동 소수점을 쓰면 실수 타입이 됨 ex) 9.46e12 = 9.46 * 10^12
3. 복소수형

### 문자열
1. 큰 따옴표와 작은 따옴표 둘 다 사용할 수 있음
2. 인접한 문자열을 하나로 합치는 기능이 있음
3. 긴 문자열 작성 방법
	1. 긴 문자열을 작성 할 때 큰 따옴표나 작은 따옴표 3개를 동시에 쓴다
	ex) """ 긴 문자열""" or ''' 긴 문자열 '''

### 그 외
1. True|False
2. 컬렉션-리스트,사전(Dictionary),집합 등

## 연산자
1. 문자열1 + 문자열2 = 문자열1 문자열2
2. 문자열1 * n = 문자열1 문자열1 문자열1 .... n번
3. 다른 타입을 연산할 때 자동 변환을 하지 않음 ex) "korea" + 2002 -> 에러/"korea" + str(2002) -> 정상 동작
4. int,str,float 타입으로 명시적 형변환을 해야함

## 조건문
**~={red}Python은 들여쓰기로 블록을 구분한다.따라서 어떤 조건에 걸리는 코드는 반드시 같은 수준의 들여쓰기여야 한다.=~** 아래 예제의 경우에 다르게 결과가 나온다
- if/if-else/if-elif-else문이 있다.
```python
time = 20
if time < 19:
	print("aaaa")
	print("bbbb")
	
if time < 10:
	print("aaaa")
print("bbbb")
```
논리 연산자 키워드: and,or,not
```python
#기본 if문
if 조건:
	명령
		
if a == 3 or b == 4:
	print("OKAY")

#if-else문
if 조건: 
	명령1
	명령2
else:
	명령3
	명령4

#if-elseif-else문
if 조건:
	명령1
elif 조건:
	명령2
else:
	명령3
```
