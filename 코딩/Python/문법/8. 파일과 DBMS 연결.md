## 파일 입출력
```python
# 작성법
open(파일경로, 모드)
file.write("작성할 내용")
file.read(n) #한번에 n 문자씩 읽음
file.readline()

#write 예시
f = open("live.txt","wt")
f.write("""작성할 문자가
많을 때 작성하는 
방법""")
f.close()

#read예시
try:
	f = open("live.txt","rt")
	text = f.read()
	print(text)
except FileNotFoundError:
	print("파일이 없습니다.")
finally:
	f.close()

f = open("live.txt","rt")
while True:
	text = f.read(10)
	if len(text) == 0: break
	print(text,end = ' ')
f.close()

# readline 예시
f = open("live.txt","rt")
text = ""
line = 1
while True:
	row = f.readline()
	if not row: break
	text += str(line) + " : " + row
	line += 1
f.close()
print(text)	
```

### 순차 접근과 임의 접근
- python은 순차 접근과 임의 접근을 지원한다ㅏ.
	- 순차 접근: 파일 객체가 다음 입출력의 위치를 정확하게 기억하고 있어서 조금씩 반복적으로 읽어도 파일 전체를 읽을 수 있는 방식(대개 처음부터 끝까지 다 읽어서 메모리로 로드하는 방식 사용)
	- 임의 접근: 파일의 원하는 부분을 자유롭게 액세스하는 방식
### 임의 접근 방식
- tell: 위치 조사
- seek: 위치 변경
	- `seek(위치,기준)`
	- 기준: 0->파일 처음부터/1->현재 위치/2->끝에서 부터
	- 위치: n->n바이트 건너뛰어서
```python
# seek 사용
f = open("live.txt","rt")
f.seek(12,0)
text = f.read()
f.close()
print(text)
```
