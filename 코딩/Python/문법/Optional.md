- `Optional`은 특정 변수가 **설정된 타입이거나 `None`일 수 있음**을 명시적으로 나타내는 타입 힌트
- `name: Optional[str]`은 name변수에 `str`이 오거나 `None(아무 값도 없음)`이 들어올수 있다고 명시

## 사용 목적
1. 명시적인 None 허용: 함수나 클래스의 변수가 None값을 가질 수 있다는 것을 명확히
2. 선택적 매개변수:함수의 매개변수가 필수가 아닐 때
3. 선택적 필드(FastAPI/Pydantic): API 요청 본문이 필수가 아닐때,클라이언트가 해당 필드를 보내지 않아도 오류 발생하지 않음
