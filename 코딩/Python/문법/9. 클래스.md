## 1. 클래스
- **~={red}생성자 작성 필수=~**
- python은 변수를 선언하는 문법이 없어 멤버 변수를 따로 선언하지 않는다. 대신 `__init__`에서 초기값을 받아 대입하는 형식으로 멤버를 생성한다
- self는 java의 this와 같은 것
- **~={red}모든 생성자와 메서드에 명시적으로 self를 인수로 넣어줘야 한다=~**.
```python
class 이름:
	def __init__(self,초기값):
		멤버 초기화
	메서드 정의

#예제
class Human:
	def __init__(self,age,name):
		self.age = age
		self.name = name
	
	def intro(self):
		print(str(self.age) + "살 " + self.name + "입니다")
		
kim = Human(29,"김상형")
kim.intro()
```

## 2. 상속
- **~={red}`class 이름(부모):`로 상속을 받는다=~**
- 생성자는 새로 추가된 멤버만 초기화 한다. **~={red}나머지는 `super.__init__(멤버)`로 초기화 한다.=~**
- 다중 상속 지원한다. 하지만 사용하지 않는 것이 바람직 하다
```python
class Student(Human):
	def __init__(self,age,name,stunum):
		super().__init__(age, name)
		self.stunum = stunum
		
	def intro(self):
		super().intro()
		print("학번: " + str(self.stunum))
	
	def study(self):
		print("공부")
```

## 3. 은폐
- 공식적으로 정보 은폐를 지원하지 않는다
- 방법
	1. getter와 setter
	2. **~={red}property(getter,setter)로 정의하여 inner 멤버에 저장=~**
		1. 멤버에 inner 붙힘
		2. @property와 @멤버.setter 이용
		3.` __`로 시작하는 변수 이용
```python
#기본적인 getter와 setter
class Date:
	def __init__(self,month):
		self.month = month
	def getmonth(self):
		return self.month
	def setmonth(self):
		if 1 <= month <= 12:
			self.month = month

#property/inner_멤버
class Date:
	def __init__(self,month):
		self.inner_month = month
	def getmonth(self):
		retur self.inner_month:
			if 1 <= month <= 12:
				self.inner_month = month
	month = property(getmonth,setmonth)
	
#property/@이용
class Date:
	def __init__(self,month):
		self.inner_month = month
	@property
	def month(self):
		return self.inner_month
	@month.setter
	def month(self,month):
		if 1 <= month <= 12:
			self.inner_month = month
			
#__이용
class Date:
	def __init__(self,month):
		self.__month = month
	def getmonth(self):
		return self.__month
	def setmonth(self,month):
		if 1 <= month <= 12:
			self.__month = month
	month = property(getmonth,setmonth)
```

## 4. 여러가지 메서드
1. ~={red}클래스 메서드=~: 클래스 전체가 공유하는 메서드 ex) 해당 클래스가 몇번 호출되었는지 세는 count 메서드
	- 클래스 메서드로 선언할 메서드 위에 `@classmethod`만 작성해주면 클래스 메서드가 됨
2. ~={red}정적 메서드=~: static메서드
	- 정적 메서드로 선언할 메서드 위에 `staticmethod`만 작성해주면 정적 메서드가 됨
3. ~={red}연산자 메서드=~: `==`,`!=`,`<`,`*`와 같은 연산자를 오버로딩해 사용자 정의대로 연산을 하게 바꿀 수 있음
4. ~={red}특수 메서드=~: 오버로딩을 통해 미리 약속된 작업을 수행함
	- `__str__`로 객체를 문자열화 함(Java의 toString()과 유사한 역할)
	- `__repr__`로 객체 표현식 만듦
	- `__len__`으로 객체의 길이 조사

### 5. 유틸리티 클래스
1. Decimal: 정확한 10진 실수를 표현하기 위해 사용
2. Fraction: 유리수 표현
3. **~={red}array=~**: 전통적인 배열/메모리 낭비가 없고 연산이 단순해 속도 매우 빠름
	- 리스트는 내부구조가 복잡하고 공간의 낭비가 심하여 속도가 느리다
	- array는 c언어 기반 타입이다
	- **`array(타입코드,[초기값])`으로 변수를 초기화 한다.**
	- ex) array('i',range(5,10))
	```python
	import array
	
	ar = array.array('i',[33,44,67,89,56])
	for a in ar:
		print(a,end = ',')
	ar.append(100)
	del ar[0]
	#등등 다양한 연산을 list보다 훨씬 빠르게 연산한다
	```
