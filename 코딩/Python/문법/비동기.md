## 비동기란
비동기: **작업의 실행과 완료를 기다리지 않고 동시에 여러 작업을 처리**
이벤트 기반이나 콜백 기반으로 구현된다.

### 파이썬의 비동기
`async`로 함수를 비동기 처리한다.파이썬에서 async를 붙힌 함수를 **~={red}코루틴=~**이라고도 부른다
```python
async def do_something():
	pass
```
- 비동기 함수를 호출하면 **coroutine(코루틴) 객체가 리턴**된다
- 비동기 함수를 다른 함수에서 호출 할 때
	1. 비동기 함수: `await` 키워드를 붙혀 호출해야 한다.
		1. ~={red}await: 코루틴 함수의 실행을 일시 중단하고 다른 비동기 작업을 하라고 요청=~
	2. 동기함수: 일반적인 함수에서 비동기 함수를 호출하려면 `asyncio.run(비동기함수())`를 이용한다.

### coroutine(코루틴)
- **메인과 서브처럼 종속 관계가 아닌 서로 대등한 관계이며 특정 시점에 상대방의 코드를 실행한다.**
- 코루틴 영역에 있는 함수가 종료되지 않은 상태에서 메인 루틴의 코드를 실행한뒤 돌아와서 코루틴의 코드를 실행 할 수 있다
- 일반 함수는 코드를 한 번만 실행할 수 있지만, 코루틴은 코드를 여러번 실행할 수 있다.
