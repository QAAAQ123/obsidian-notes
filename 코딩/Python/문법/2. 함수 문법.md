## 매개변수
### 1. 가변 매개변수
- 임의 개수의 매개변수를 한개의 변수로 받을 수 있다
- `*args`로 매개변수에 asterisk를 붙힌다.
- 가변 매개변수는 반드시 **~={red}매개변수 목록의 마지막에 딱 하나만=~** 나와야 한다.
	- `intsum(s,*ints)`는 가능하지만 `intsum(*ints,s)`와 `intsum(*ints,*nums)`는 불가능하다
```python
def intsum(*ints):
	sum = 0
	for num in ints:
		sum += sum
	return sum
```
### 2. 매개변수 기본값 지정
- 매개변수 뒤에 `= 값`을 붙히면 사용하는 곳에서 값을 지정하지 않아도 기본값이 들어간다.
- 기본값을 가지는 매개변수는 **~={red}반드시 매개변수 목록 뒤쪽에 적어=~**야 한다.
	- `intsum(begin,end=10,step=1)`은 가능하고 `intsum(begin,end=10,step)`는 불가능하다
```python
def calcstep(begin,end,step=1):
	sum = 0
	for num in range(begin,end + 1,step):
		sum += sum
	return sum
	
print(calcstep(1,10,2))
print(calcstep(1,10))
```

### 3. 키워드 매개변수
- 매개변수의 개수가 많아지면서 변수의 순서를 헷갈리는 것 방지
- 키워드 매개변수는 반드시 맨 뒤 쪽에 와야한다.
```python
print(a,b)
print(a,b,sep=',')
print(a,b,sep='',end='$')
print(a,b,end='$',sep='')
```

### 4. 키워드 가변 매개변수
- `**args`로 asterisk를 2개 붙힌다
- dictionary 타입으로 함수 내부에 들어가서 사용한다
```python
def calcstep(**args):
	begin = args['begin']
	end = args['end']
	step = args['step']
	
	num = 0
	for num in range(begin,end + 1,step):
		sum += num
	return sum
	
print(calcstep(begin=3,end=8,step=1))
print(calcstep(step=2,begin=1,end=10))
```
