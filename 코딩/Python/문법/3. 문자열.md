## 포맷팅
1. C의 printf와 같은 방식으로 동작/문자열 뒤에 `%(변수)`형태로 집어 넣는다.
	ex) `print("%d월 %d일은 %s이다." %(8,15,"광복절")`
2. 선형 포맷팅 방식: {} 사용해 포맷팅/문자열 뒤에 `.format(변수)`형태로 집어 넣는다
	1. 변수의 순서가 같은 경우: `print("이름{}, 나이{}, 키{}".format(name,age,height))`
	2. 변수의 순서가 다른 경우: `print("이름{2}, 나이{0}, 키{1}".format(age,height,name))`
	3. 변수 키워드로 집어 넣을 경우 `print("이름{name}, 나이{age}, 키{height}".format(name = "한결",age=20,height=160.9))`
