## Pull request란?
협업자에게 branch 병합을 요청하는 메시지
GitHub에서만 가능한 기능

### Pull request의 브랜치 설정
base branch: 병합 결과물이 저장될 branch
compare branch: 병합할 대상인 branch
Github 화면에서는 base: xxxx <- compare: xxxx 이런식으로 나온다.

Github의 pull request 입력창
1. Reviewers: 검토를 요청할 협업자 지정
2. Assigness: pull request 담당자
3. Labels: pull request 태그 설정
   ex)[버그],[프런트엔드],[백엔드] 등...
