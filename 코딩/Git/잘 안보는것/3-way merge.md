브랜치간 병합을 효과적으로 해결하기 위한 병합 전략

## 병합 기반
1. 두 브랜치의 공통 조상
2. 현재 브랜치의 마지막 커밋
3. 병합할 브랜치의 마지막 커밋

## 병합 과정
1. 공통 조상 커밋을 기준으로 어떤 파일이 어떻게 변경되었는지 확인
2. 병합
	1. 한쪽에서만 변경된 파일을 자동 병합
	2. 두쪽에서 변경되지 않은 파일은 그대로 유지
3. 충돌 
	1. 두 브랜치에서 같은 라인이 수정되었을 경우, 사용자가 수정하여 충돌 해결후 병합


#### 3-way merge 사용 이유
공통조상을 기반으로 진짜 변경사항만 식별하여 병합
정확하고 효율적인 병합
