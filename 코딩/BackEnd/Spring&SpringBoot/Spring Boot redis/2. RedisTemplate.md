### RedisTemplate란?
- Redis의 기본 자료구조 5개에 대응하는 오퍼레이션 메서드를 제공하여 데이터 제어

### 자료구조별 제공 메서드
| **메서드**         | **Redis 자료구조** |
| --------------- | -------------- |
| `opsForValue()` | **String**     |
| `opsForList()`  | **List**       |
| `opsForSet()`   | **Set**        |
| `opsForZSet()`  | **Sorted Set** |
| `opsForHash()`  | **Hash**       |

### 자료구조별 주요 메서드
| 오퍼레이션 메서드       | 인터페이스             | 주요 메서드                  | 매핑 Redis 명령어   |
| --------------- | ----------------- | ----------------------- | -------------- |
| `opsForValue()` | `ValueOperations` | `set()/get()`           | `set/get`      |
| `opsForList()`  | `ListOperations`  | `leftPush()/rightPop()` | `lpush/rpop`   |
| `opsForSet()`   | `SetOperations`   | `add()/members()`       | `sadd/smember` |
| `opsForHash()`  | `HashOperations`  | `put()/get()`           | `hset/hget`    |
| `opsForZSet()`  | `ZSetOperations`  | `add()/range()`         | `zadd/zrange`  |
