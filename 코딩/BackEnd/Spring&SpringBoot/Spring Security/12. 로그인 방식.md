## Spring Security로 구현 할 수 있는 로그인 방식의 종류
1. formLogin -> 이미 했음(HTML Form 태그)
2. HttpBasic

[HttpBasic 인증 방식 공식 문서](https://docs.spring.io/spring-security/reference/servlet/authentication/passwords/basic.html)
## HttpBasic 인증 방식
- 브라우저에 id,pw를 넣어서 인증하는 방식
- Http Basic 인증 방식은 아이디와 비밀번호를 Base64 방식으로 인코딩한 뒤 HTTP 인증 헤더에 부착하여 서버측으로 요청을 보내는 방식
- SecurityConfig.java 파일에서 formLogin() 메소드 대신에 httpBasic() 메소드를 사용한다.
```java
@Bean
public SecurityFilterChain filterChain(HttpSecurity http) throws Exception{

    http
            .httpBasic(Customizer.withDefaults());

    return http.build();
}
```
