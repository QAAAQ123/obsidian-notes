- Spring Security는 사용자가 로그인 할 시에 **아이디와 비밀번호를 검증**한다.
- 이때 비밀번호는 ~={red}**단방향 해쉬 암호화**=~를 시켜 DB에 저장되어 있는 비밀번호와 ==대조==한다.
- Spring Security는 단방향 암호화 중에 **BCrypt Password Encoder 사용 권장**
- 따라서 해당 클래스를 return하는 메소드를 만들어 @Bean으로 등록하여 사용하면 된다.
- ~={red}**단방향 암호화: 암호화만 가능하고 복호화는 불가능한 방식**=~
```java
@Bean  
public BCryptPasswordEncoder bCryptPasswordEncoder() {  
    //암호화 메소드 구현 완료 + Bean에 등록
    return new BCryptPasswordEncoder();  
}
```
