## 현재 위치 가져오기
1. 채팅 보낼 때 좌표 같이 실어 보내기
2. 좌표만 주기적으로 보내면 Redis에서 저장하는 방식
3. FastAPI에서 직접 수신(보안 처리를 중복해야함)


### 1. 위치 관리 기능
- 사용자 실시간 GPS 좌표(위경도) 수집 기능
- 위치 탐색 실패 및 권한 거부 시 예외 처리 기능

### 2. 지도 및 경로 시각화 기능
- 추천 경로 옵션(경로 A, B 등) 선택 및 전환 기능
- 지도 위 번호형 장소 마커 표시 기능
- 마커 간 이동 경로 선(Polyline) 표시 기능

### 3. 경로 편집 기능 (팝업/바텀시트)
- 경로 렌더링 시 편집 팝업 자동 호출 기능
- 현재 선택된 경로의 장소 리스트 노출 기능
- 개별 장소 삭제(-) 기능
- 하단 대체 추천 장소 목록 노출 및 추가(+) 기능
- 장소명 터치 유지 시(Long-press) 상세 정보 미리보기(모달/툴팁) 기능
- 코스 확정(완료/저장) 기능
- **예외 처리:** 모든 장소 삭제 시도 방지(최소 개수 유지) 기능

### 5. 여행 코스 상세 탭 기능
- 확정 시 '여행 코스' 탭 화면 자동 전환 기능
- 확정된 장소 리스트 렌더링 기능 (장소명, 주소 노출)
- 외부 API 연동을 통한 장소 썸네일 이미지 및 추가 상세 정보 호출 기능
- 장소 연관 외부/내부 딥링크(Deep Link) 이동 기능

### 6. 서버 및 AI 추천 로직 (백엔드)
- AI 프롬프트 기반 위치/테마 맞춤형 장소 추천 기능
- 시간대 기반 추천 우선순위(음식점 vs 관광지) 배치 로직
- 장소 최대/최소 허용 개수 및 카테고리 비율 제어 로직
- 추천할 장소가 없을 시 반경 확장 또는 예외 메시지 반환 기능
- 최종 확정된 여행 코스 DB 저장 기능
