## 현재 위치 가져오기
1. 채팅 보낼 때 좌표 같이 실어 보내기
2. 좌표만 주기적으로 보내면 Redis에서 저장하는 방식
3. FastAPI에서 직접 수신(보안 처리를 중복해야함)



### 📍 1. 위치 및 권한 (Location & Permission)
- **현재 위치 수집 기능:** 브라우저/디바이스를 통한 사용자 실시간 위경도 좌표 획득

### 💬 2. 채팅 및 AI 추천 (Chat & AI Recommendation)
- **채팅 입력 및 전송 기능:** 코스 추천 요청을 위한 텍스트 입력창 및 전송 UI
- **빈 화면 리다이렉트 기능:** '여행 코스' 탭에 확정된 코스가 없을 때 `[코스 생성하기]` 버튼 노출 및 채팅 화면 강제 이동
- **AI 로딩 상태 표시 기능:** 코스 생성 대기 시간 동안 시각적 피드백(로딩 스피너/메시지) 제공
- **조건부 AI 추천 로직 (백엔드)**
        

### 🗺️ 3. 지도 및 경로 시각화 (Map & Routing UI)
- **커스텀 마커 표시 기능:** 추천된 장소 위치에 방문 순서(숫자)가 표기된 마커 노출
- **이동 경로(Polyline) 렌더링 기능:** 마커 간의 이동 동선을 선으로 연결하여 시각화
- **다중 경로 전환 기능:** 탭 또는 버튼을 통한 추천 경로 옵션(경로 A, 경로 B) 간 전환

### ✍️ 4. 경로 편집 팝업 (Course Edit)

- **팝업/바텀시트 자동 호출 기능:** 지도에 마커가 렌더링됨과 동시에 편집 UI 자동 활성화
    
- **장소 목록 렌더링 기능:** 현재 선택된 경로의 장소를 순서대로 리스트 형태로 표시
    
- **장소 삭제 기능 (`-`):** 특정 장소를 현재 코스에서 제외
    
- **장소 추가 기능 (`+`):** 하단 대체 추천 장소 목록에서 새로운 장소를 코스에 추가
    
- **동선 재정렬 기능:** 장소 추가/삭제 시 지도 위 마커 번호와 연결 선(Polyline) 즉시 업데이트
    
- **코스 확정 기능:** `[완료/저장]` 버튼을 통한 경로 최종 확정 및 탭 전환
    

### 🔍 5. 상세 정보 확인 (Detail Info View)

- **팝업 내 상세 정보 미리보기 기능:** (아코디언 확장, 롱프레스, 전체 슬라이드 전환 중 택 1 구현)
    
- **마커 연동 정보창(InfoWindow) 기능:** 지도 위 마커 터치 시 장소 요약 팝업 표시
    
- **여행 코스 상세 리스트 생성 기능:** 확정된 코스의 장소명, 썸네일, 주소, 상세 정보 등을 카드 형태로 나열
    
- **외부 앱 딥링크(Deep Link) 기능:** 상세 정보 내 링크 클릭 시 외부 길찾기(카카오맵/네이버 지도) 또는 웹사이트로 즉시 연결
    

### 💾 6. 데이터 및 세션 관리 (Data & Session Management)

- **코스 재수정 기능:** '여행 코스' 탭에서 `[수정하기]` 버튼을 눌러 다시 지도/팝업 편집 화면으로 복귀
    
- **임시/확정 데이터 분리 저장 기능:** 편집 중인 데이터(Redis)와 최종 확정된 코스 데이터(DB)의 분리 처리
    
- **코스 유지 기간(TTL) 관리 기능:** 설정된 시간(예: 24시간) 경과 시 임시 저장된 코스 데이터 자동 초기화
    

### 🚨 7. 예외 및 오류 처리 (Error Handling)

- **위치 탐색 실패 처리:** 권한 거부 또는 GPS 오류 시 대체 입력 유도 메시지 노출
    
- **주변 장소 부족 처리:** 기준 반경 내 추천 장소가 부족할 경우 안내 메시지 노출 및 탐색 반경 자동 확장
    
- **최소 장소 개수 방어 로직:** 사용자가 모든 장소를 삭제(-)하려고 할 때 경고 모달 노출 및 삭제 방지
    
- **AI/서버 통신 실패 처리:** 타임아웃 또는 서버 에러 발생 시 "경로 생성 실패" 안내 및 재시도 유도
