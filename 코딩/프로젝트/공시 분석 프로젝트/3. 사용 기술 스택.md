## 웹 서버 및 프레임워크
1. fastapi: 비동기 웹 프레임워크.
2. uvicorn: ASGI 서버 구현체(웹 서버)
	1. 애플리케이션 코드가 실제 네트워크 상에서 동작할 수 있도록 실행
	2. HTTP 통신을 처리
3. Jinja2: 템플릿 엔진-동적 웹 페이지 렌더링
4. python-multipart: 폼 데이터 파싱 라이브러리
	1. HTML Form 태그에서 전송된 데이터를 FASTAPI가 읽을 수 있도록 변환
5. pydantic: 데이터 유효성 검사 라이브러리
	1. 데이터 타입 검증,스키마 정의

## AI 및 데이터 수집
1. google-generativeai: Google Gemini API를 사용하기 위한 공식 Python 클라이언트 SDK
2. requests: HTTP 클라이언트 라이브러리
	1. HTTP 요청,응답 
3. beautifulsoup4: HTML/XML Parser
	1. request로 받아온 HTML 문서의 구조를 분석하여 특정 태그나 텍스트 데이터 추출

## 데이터베이스
1. SQLAlchemy: ORM 라이브러리
	1. 객체 사용해서 DB테이블 생성 및 조작
