1. 비식별/식별 관계 파악
2. 선택/필수 참여 파악
3. 다중값 관계 파악

- 유저-글 관계
	- 유저는 글이 없어도 된다.(선택 참여)
	- 글은 반드시 유저가 있어야 한다(필수 참여) -> 유저가 삭제되어도 글이 남아있는 것은 외래키 제약조건으로 관리
	- 유저-글은 비식별 관계이다. 
	- 유저 -> 글은 1:N 관계
	- 글 -> 유저는 N:1 관계
	- 정리: 유저-선택참여,1:N/글-필수참여,N:1/비식별
	- 유저쪽 O / 글쪽 |와 새발 / 점선
- 글-미디어 관계
	- 글-선택 참여
	- 미디어-필수 참여
	- 글-미디어는 식별관계이다
	- 글-> 미디어는 1:N 관계
	- 미디어 -> 글은 N:1 관계
	- 정리: 글-선택참여,1:N/미디어-필수참여,N:1/식별
	- 글쪽 O / 미디어쪽 |와 새발 / 실선
- 글-댓글 관계
	- 글-선택 참여
	- 댓글-필수 참여
	- 글-댓글은 식별관계이다
	- 글-> 댓글은 1:N 관계
	- 댓글 -> 글은 N:1 관계
	- 글쪽 O / 댓글 쪽 |와 새발 / 실선
- 유저-댓글 관계
	- 유저-선택 참여
	- 댓글-필수 참여 -> 댓글 남아있는거 제약조건으로 관리
	- 유저-댓글은 비식별 관계이다
	- 유저->댓글은 1:N 관계
	- 댓글->유저는 N:1 관계
	- 정리: 유저-선택참여,1:N/댓글-필수참여,N:1/비식별
	- 유저쪽 O / 댓글쪽 |와 새발 / 점선
