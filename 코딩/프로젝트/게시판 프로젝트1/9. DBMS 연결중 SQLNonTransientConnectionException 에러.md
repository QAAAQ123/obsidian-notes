- 에러 내용
	- Caused by: java.sql.SQLNonTransientConnectionException: Public Key Retrieval is not allowed

1. DBMS 사용자에게 해당 DB 권한 부여: board_service DB에 관한 모든 권한 부여 - 해결 안됨
2. 사용자 권한 확인 : show grants for '사용자'@'locahost'; - board_service에 대한 모든 권한 부여 확인
3. spring.datasource.url에 allowPublicKeyRetrieval=true 추가- 해결(**배포시에 SSL은 true로 allowPublicKeyRetrieval은 flase로 변경해주어야 한다**)

MySQL 8.0 이상 버전에서 기본 인증 방식이** `caching_sha2_password`**로 바뀌면서** 발생하는 문제
