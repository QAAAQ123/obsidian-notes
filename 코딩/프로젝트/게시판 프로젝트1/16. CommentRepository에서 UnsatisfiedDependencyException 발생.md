```java
@Query("SELECT c.postId FROM Comment AS c WHERE c.commentContent LIKE %:keyword%")  
List<Long> searchAllPostIdByKeywordAtComment(@Param("keyword") String keyword);
```
위 코드에서 UnsatisfiedDependencyException이 발생했다. 
이유: JPQL은 DB가 아닌 Entity의 관점에서 접근해야 한다.그리고 Comment entity에는 postId가 아닌 post가 있고, 그 post안에 postId가 있다. 따라서 c.postId -> c.post.postId로 수정했더니 에러가 나오지 않았다.
배운것: JPQL 데이터베이스의 **테이블**이 아닌 **엔티티 객체**를 대상으로 쿼리를 작성하는 객체지향 쿼리 언어
