인공지능/머신러닝과 딥러닝은 다르다
인공지능이란? 인간처럼 생각하고 판단할 수 있도록 만들어진 컴퓨터 프로그램

인공지능>머신러닝>딥러닝 포함관계이다
인공지능을 만드는 한가지 분야가 딥러닝과 머신러닝이다.


## 머신 러닝
- 데이터로 패턴 학습하여 예측,결정하는 기술
- 특징 설계 과정이 필요(사람이 직접 특징 추출 방법을 설계)
- 직접 사람이 설계해야 한다. 고양이와 개의 분리를 할때 사람이 직접 귀의 모양,눈의 크기,털 색깔 등 직접 사람이 어떤 특징을 사용해서 구별할지 지정하는 방식 -> 전문가에게 의존해야 하기 때문에 자동화가 어려움/특징을 정리하기 어려운 경우에 사용하기 어려움/하나에 구별을 할 때마다 새로운 특징을 제공해줘야 한다.

## 딥러닝
- 머신러닝의 한 분야
- Deep Neural Network(깊은 신경망)과 표현 학습(Representation Learning)을 합친 것이다
- **반드시 신경망을 사용**해야 하고 신경망이 자동으로 특징을 추출한다.
- 특징 추출 방법 자체를 컴퓨터가 자동으로 생성하여 학습한다. 고양이와 개를 구분할 때와 개와 늑대를 구별할 때 다른 특징을 자동으로 추출하여 학습한다.

머신러닝은 사람이 직접 특징을 뽑아서 해야해서 자동화가 어렵고 전문가에 의지할 수 밖에 없다. 다른 도메인에서 적용되지 않을 수 있다.
딥러닝은 어떤 특징을 추출할지 결정한다.
## 퍼셉트론
- 신경망을 구성하는 기본 단위
- 인간의 뉴런에서 영감을 얻음
- 인공 뉴런 모델

## 어떤 분야에서 활용할 수 있을지(딥러닝)
1. 강화학습: 시스템에 지속적으로 보상을 주면서 시스템이 스스로 발전해 나갈 수 있도록 발전해나가는 기술
2. 자연어처리: 텍스트 분류,번역,요약,감정 분석 등 여러가지 task들이 자연어 처리에 포함됨/ 최근에 LLM 기술이 많이 발전해서 분리되어있던 기술들을 한 곳에서 함
3. 컴퓨터 비전: 이미지,비디오 등과 같은 시각적 정보를 처리하는 것
4. 음성 처리: 대화나 오디오 정보 처리(음성인식,노이즈 인식,대화 or 오디오 정보 생성)
