## Login 로직
- login
	- security configuration
		- auth manager??
		- 비밀번호 인코더
		- **~={red}filter chain=~**
			- csrf:disable(기본 로그인을 차단하기 때문에 필요 없음)
			- httpBasic: disable(security기본 로그인 차단)
			- HttpRequest: 모든 요청 허용
			- cors: 기본 cors옵션 적용
			- 예외
				- 진입포인트: 인증안되어 있으면(인증x) 401 에러
				- 거절 핸들링: 접근 거절이면(권한부족) 403 에러
			- FilterBefore: 로그인 전 Token검사,Token없으면 LoginFiler로 넘김
			- FilterAt: 매니저를 기본 인증 매니저에서 내가 만든 매니저로 바꿈
			- sessionManagement: 기본 상태인 JSESSIONID세션 저장을 세션을 저장하지 않게 바꿈
		- JWT OncePerRequestFilter의 doFilterInternal: Spring에서 커스텀 필터를 만들 때 핵심 로직을 작성하는 메서드 #doFilterInternal #filter
