## 기능 추가 할 것
### 지도
- GET: 음식점/카페/관광지 버튼을 누르면 현재 사용자 반경 수m이내의 음식점/카페/관광지 데이터 JSON 전송
- GET: 마커를 누르면 해당 장소의 세부 정보 전송-이때 찜목록에 kakaoPlaceId와 비교해서 하트를 결정해야함
- POST: 찜한 장소로 추가(xy좌표, 카카오placeid,장소이름,주소,평점,카카오맵URL은 kakaoplaceid로 해결가능)
- GET: 경로 버튼 누르면 경로에 해당하는 순서대로 X,Y좌표와 세부 정보 JSON 전송
- GET: 마커 얼굴 사진 URL JSON으로 전송

### 채팅
채팅이 길어지면 세션 새로 만들기 or 채팅이 길어져도 계속 한 세션으로 유지 or 정해진 주기에 따라서 세션 새로 만들기
- GET: 추천 검색어(해시태그) JSON 전송
- GET: 이전 대화 내역 불러오기(한번에 불러오는 양 얼마나 할 것인지)
- POST: 사용자가 전송을 누르면 답변 JSON전송(~={red}답변 보내는 방식????=~)
- 첫번째 질문 입력하면 제목 생성하는 기능

- MyInfo화면에서
	- DELETE: 채팅세션 삭제 기능

- POST: 새로운 채팅 세션 생성하는 기능
