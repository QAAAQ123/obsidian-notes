전체 음식점-사상의학에 맞는 음식점 search
MBTI와 사상의학을 상업적으로 연결 간읗함
가자곰 연공은 협의 해야함 + 소상공인도 발급 가능한 상품권(카카오톡 선물하기)서비스 연동
가자곰 연동: 팬텀지갑,가자곰으로 지갑 연동-솔라나로 결제

**반응형 웹으로**

비즈니스 모델과 접목
추천 경로 수정,삭제,추가

------------ 정리 --------------------
- 사용자 얼굴 기반의 아바타 생성 및 셀카 기능 추가
- AWS AI를 어쩔수 없이 사용해야 하는데, LLM을 Chat GPT로 변경하면 성능이 매우 좋아지고 앱 자체의 평판도 좋아질 것이다
- 추천 경로 탭에 카페나 음식점의 이벤트 정보를 크롤링하여 넣을 수 있으면 좋겠다 -> 추후에 크롤링 대신 광고를 받아서 수익화 가능
- 전체 음식점 토글 버튼으로 MBTI와 사상의학에 맞는 음식점을 찾아주면 좋겠다
- MBTI와 사상의학을 상업적으로 이용 가능하다
- 가자곰 연동은 아직 협의되지 않았다.
	- 가자곰: 팬텀 지갑을 가자곰과 연동,결제
	- 솔라나는 암호화폐 및 스마트 계약 플랫폼이고 팬텀은 솔라나 블록체인 기반의 대표적인 지갑
- 가자곰 연동 대신에 '내기프트'를 연결할 수 있도록 상의해보겠다
	- [이 사이트 같음](https://www.naegift.com/)
- 앱으로 하면 IOS,안드로이드 둘 다 해야한다. 시간과 자원의 한계로 IOS를 못하기 때문에 반응형 웹이 더 적절해 보인다. 
- 안드로이드 앱은 추후 발표할 때 왜 안했냐고 질문이 나올 가능성이 크다. 
	- 프로토타입 형태로 1,2개의 탭을 만들고 '이렇게 구현 가능하다'정도만 보여주면 될 것 같다
- 앱은 보안 문제가 있다
- 비즈니스 모델과 연계할 수 있는 쪽으로도 생각해보는게 좋겠다. 
- 추천 경로를 만들때 수정 기능 뿐만 아니라 삭제,추가도 할 수 있게 만들면 좋겠다
