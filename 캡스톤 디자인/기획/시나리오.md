1. 사상의학의 체질 4가지 X MBTI 16가지 총 64가지의 추천경로 주기적 생성
2. 사용자가 사상의학과 MBTI에 관한 간단한 질문에 대답 -> 대답한 정보를 바탕으로 사용자의 체질,MBTI 정보 출력 -> 이미 만들어진 코스를 사용자의 체질과 MBTI에 맞게 제공(예시: 소양인 + I에게는 약쑥 뜸 체험,전등사 찾집에서 쉼)
3. 추천 코스대로 여행을 한다고 가정 할 경우
	1. QR이나 GPS+사진 인증 기반으로 해당 관광지에 방문했다는 NFT를 발급
	2. 발급 받은 NFT를 클릭하면 해당 관광지에 대한 정보 제공(+ LLM을 이용한 특정 페르소나를 기반으로 한 텍스트 생성 추가 가능)
	3. 한 관광지 여행을 마치고 다음 관광지로 이동할 때 거리가 멀 경우 뭐를 타는게 가장 빠르고 편리한지 정보를 제공해주는 길찾기(인팁 앱과 비슷한 기능)
	4. 예약이 필요한 관광지일 경우 예약 사이트로 연결
4. 추천 코스를 따라가지 않는 다고 가정 할 경우
	1. 사용자의 조사 결과와 가장 비슷한 경로 추천 -> 3으로 돌아감
	2. 사용자가 가고 싶은 관광지 2-3곳을 선택하거나 키워드 입력
	3. 그 주변에 음식점,카페나 추가 장소를 제공

AI 범위: 추천 코스 생성,맛집 추천,길찾기, 주변 음식점,카페,추가장소 제공

### NFT로 활용 가능 한 것들
1. NFT를 발급 할 때 사용자가 찍은 사진으로 NFT의 아이콘을 만드는 것
2. 장소 기반의 '디지털 영토 증명 NFT'
	- **핵심 아이디어**: 인천의 주요 관광지나 숨겨진 명소들을 수백 개의 작은 구역(Hexa)으로 나눈 디지털 지도를 만듭니다. 관광객이 특정 구역에 물리적으로 방문하여 미션을 완료하면, 해당 구역의 '디지털 오너십(소유권)'을 증명하는 NFT를 발행받습니다.
- NFT를 보여주면 할인 쿠폰처럼 사용 가능하게
- 여행을 마친 관광객이 자신의 동선과 팁을 하나의 NFT로 만들어 판매 할 수 있는 거
- 소지품 등록 NFT를 만들어 관광객의 소지품을 분실 하였을 경우 사용자가 분실 신고하면 NFT의 상태가 '분실'로 변경되며, 경찰/유실물 센터/호텔 등에 바로 공유하여 쉽게 찾을 수 있게
