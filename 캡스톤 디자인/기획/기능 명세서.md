### 1. 가입 화면
- **회원가입 기능:** '회원가입' 버튼을 통해 새로운 계정을 생성하는 기능입니다.▲
- **로그인 기능:** '로그인' 버튼을 통해 기존 회원이 서비스에 접근하는 기능입니다.▲
- **Google 간편 로그인 기능:** Google 계정을 연동하여 별도 가입 절차 없이 간편하게 로그인하는 소셜 로그인 기능입니다.
- **닉네임 입력 기능:** 사용자의 닉네임을 입력받는 기능입니다.
- **생년월일 입력 기능:** 사용자의 나이(생년월일)를 입력받는 기능입니다.
- **체류일수 입력 기능:** 사용자가 얼마나 머무를지를 입력받는 기능입니다.
- **MBTI 입력 기능:** 사용자의 MBTI 유형을 입력받는 기능입니다.
- **사상의학 테스트 기능:** '사상의학 테스트' 버튼을 눌러 사용자의 체질을 진단하는 별도 테스트를 진행하는 기능입니다.
- **여행 동반자 선택 기능:** 사용자가 '혼자' 여행하는지, '친구'와 여행하는지 등 동반자 유형을 선택하는 기능입니다.
- **페르소나 선택 기능:** 사용자가 동행할 'AI 페르소나'를 선택하는 기능입니다.
- **가입 완료 기능:** '완료' 버튼을 눌러 입력된 모든 프로필 정보를 서버에 저장하고 가입을 마치는 기능입니다.
- **키보드 연동 및 활성화/비활성화 기능:** 텍스트 입력창 터치 시 키보드가 올라오고, 입력 완료나 다른 영역 터치 시 사라지는 기능입니다.

### 2. 메인 화면(지도)
- **네이티브 지도 연동 기능 (필수):** 앱의 90%를 차지하는 지도를 화면에 띄우는 기능입니다. (Naver/Kakao 지도 API 연동)
- **현재 위치 표시 기능:** 지도 위에 사용자의 현재 GPS 위치를 파란 점 등으로 표시합니다.
- **AI 페르소나 표시 기능:** 사용자가 선택한 AI 페르소나 아이콘을 지도 위에 항상 떠 있도록(오버레이) 표시합니다.
- **페르소나 상태 변경 기능:** 사용자의 상황이나 기분에 따라 AI 페르소나 아이콘의 표정 등이 변하는 기능입니다.▲
- **능동적 추천 기능 (핵심):** AI가 사용자의 위치/성향에 따라 특정 장소를 붉은색 핀으로 찍어 선제적으로 제안합니다.
- **주변 소(POI) 표시,지도 핀 터치,주변 평점 정보 표시,장소 정보 표시 기능**: 구글 API 사용해 구현
- **간략 정보창(팝업) 기능:** 지도 핀 클릭(터치) 시 해당 장소의 이름, 평점, 간략 정보를 보여주는 작은 팝업창을 띄웁니다.
- **해시태그 ON/OFF 기능:** 해시태그를 활성화하면 관련 장소(음식점/카페/관광명소)를 모두 지도에 표시하는 필터 기능입니다.
- **여행 코스 ON/OFF 기능:** 하루치 여행 코스를 지도에 마커와 선으로 표시하거나 숨길 수 있는 토글 기능입니다.
### 3. 채팅 화면 (AI 인터페이스)
- **AI 코스 생성 기능 (핵심):** 개인화 정보(MBTI, 체류일수 등)를 바탕으로 AI가 맞춤형 여행 코스를 생성한후 사용자의 피드백을 받아 최종적으로 경로를 결정하는 기능입니다
- **채팅창 열기/닫기 기능 (슬라이드 업 패널):** 메인 화면 하단에서 채팅창을 위로 쓸어 올리거나(Slide-up) 핸들바를 드래그하여 닫는 기능입니다.
- **텍스트 메시지 입력 기능:** 하단 입력창을 통해 사용자가 AI에게 직접 질문이나 요청을 텍스트로 작성하는 기능입니다.
- **메시지 전송 기능:** 입력창 우측의 화살표 버튼을 눌러 작성한 메시지를 AI에게 보내는 기능입니다.
- **AI 응답 수신 및 표시 기능:** AI의 답변 내용을 말풍선 형태로 화면에 표시하는 기능입니다.
- **추천 키워드/태그 선택 기능:** '#쇼핑', 'MBTI' 등 미리 제시된 키워드 버튼을 터치하여 AI에게 정보를 전달하거나 질문하는 기능입니다.
- **대화 기록 스크롤 기능:** 현재 채팅창 내에서 이전 대화 내용을 확인하기 위해 화면을 스크롤하는 기능입니다.
- **키보드 연동 및 활성화/비활성화 기능:** 텍스트 입력창 터치 시 키보드가 올라오고, 입력 완료나 다른 영역 터치 시 사라지는 기능입니다.


### 4. 코스 생성 및 코스 정보 제공 (추천 경로 탭)
- **가입 완료 후 자동 코스 생성 기능:** 회원가입 시 입력된 정보를 바탕으로 여행 코스를 자동으로 생성하여 '추천 경로' 탭에 코스에 대한 자세한정보를 제공하는 기능입니다.
- **코스 확정/수정 기능:** AI가 제안한 코스를 사용자가 확정하여 저장하거나, 수정을 요청하여 재구성하는 기능입니다.


### 5. 길찾기 화면
- **출발지/목적지 입력 기능:** 사용자가 텍스트로 출발지와 목적지를 직접 입력하거나, 지도에서 선택하여 설정하는 기능입니다.
- **출발지/목적지 전환(Swap) 기능:** 화살표 아이콘(↑↓)을 터치하여 출발지와 목적지를 서로 맞바꾸는 기능입니다.
- **경로 검색 요청 기능:** 설정된 구간의 이동 경로를 탐색하는 기능입니다.
- **경로 목록 표시 기능:** 검색된 다양한 경로 옵션(버스, 지하철 등)을 소요 시간 등과 함께 리스트로 보여주는 기능입니다.
- **상세 경로 안내 기능:** 선택한 경로의 환승 정보, 도보 구간 등 세부 이동 과정을 단계별로 보여주는 기능입니다.
- **지도 위 경로 표시 기능:** 선택한 이동 경로를 지도 위에 선으로 시각화하여 보여주는 기능입니다.
- **교통 수단 선택 기능:** 버스, 지하철, 자가용, 도보 등 원하는 이동 수단별로 경로를 필터링하는 기능입니다.
- **키보드 연동 및 활성화/비활성화 기능:** 텍스트 입력창 터치 시 키보드가 올라오고, 입력 완료나 다른 영역 터치 시 사라지는 기능입니다.
    

### 6. MyInfo 화면
- **채팅 기록 확인 기능:** AI 메이트와 나누었던 지난 대화 내역 전체를 별도 화면에서 모아보는 기능입니다.
- **찜한 장소 확인 기능:** 사용자가 지도나 추천 목록에서 '찜' 해둔 장소들을 모아 목록으로 보여주는 기능입니다.
- **가자곰(프로필/캐릭터) 관리 기능:** 내 페르소나(AI 메이트)의 상태를 확인하거나 관련 설정을 하는 기능입니다.
