## 1. 서비스 정의: 한국적 요소를 이용한 사용자 맞춤형 여행 정보 제공 서비스

1. 문제 정의(통계 자료 넣으면 좋을 것 같음)
	1. 문제: 크루즈나 공항을 통해 인천으로 들어오는 (단기 여행)관광객들이 많지만, 인천의 관문도시로서의 인식과 관광지로서의 매력도가 부족하여 인천 관광을 포기하는 경우가 많다. 정보가 많고 여행 매력도가 높은 서울이나 경기도로 이동하여 관광을 한다
	2. 기존 해결 방식: '인천e지'와 같은 앱을 이용하여 인천을 여행하거나 서울/경기로 이동하여 여행을 한다
2. 타겟 고객
	1. 메인 타겟: 인천 크루즈 터미널이나 인천 공항을 통해 들어오는 (단기)관광객을 타겟으로 한다
	2. 특징: 관광에 대한 의지는 높으나, 인천의 관문 도시로서의 인식 및 부족한 여행 인프라로 인해 인천 관광을 기피하고 있는 상황이다
3. 핵심 가치 및 솔루션
	1. 핵심 가치: 인천 여행을 하지 않는 관광객들이 인천을 여행하게 전환시키는 것. 한국 문화를 기반의 맞춤형 정보로 인천에 대한 즐거운 경험을 제공한다
	2. 솔루션: 한국의 독특한 문화인 MBTI와 사상의학과 생성형 AI를 바탕으로 개인에게 맞는 여행 코스를 추천하고, 여행지에서 NFT를 발행하여 여행을 기록하고 NFT 기반으로 다양한 혜택을 제공한다
4. 주요 기능
	1. MBTI와 체질(사상의학) 조사: 사용자의 MBTI와 체질을 간단한 질문으로 결정한다
	2. 코스 추천: MBTI와 체질별로 생성형 AI를 이용해 코스를 생성 해둔 후, 사용자에게 맞는 코스를 제공한다.
	3. 여행 정보 통합
		1. 여러곳에 퍼져있는 여행 정보를 통합하여 한 곳에서 모든 정보 제공
		2. 여행지에 있는 QR 코드를 찍으면 해당 여행지의 정보 제공한다
	4. 여행지 기록: NFT를 발행하여 관광객이 해당 여행지를 방문하였음을 기록한다
5. 차별점
	1. 한국 문화의 부각: 한국에서만 유행하는 MBTI와 한국의 고유 의학인 사상의학을 여행 코스 추천의 기반으로 함으로써 외국인 관광객이 한국 문화를 접할 수 있게 한다
	2. 인천 특화: 인천국제공항과 크루즈 터미널을 중심으로 한 인천의 지리적·문화적 자산에 집중하여, 다른 지역 정보와 분리된 깊이 있는 인천 맞춤형 여행 경험을 제공한다
	3. 개인 맞춤형 여행 코스: MBTI(성격 유형)와 사상의학(체질)이라는 독특한 기준을 융합하여, 사용자의 내면적 성향과 신체적 컨디션까지 고려한 세상에 하나뿐인 여행 코스를 제안한다
	4. 여행 정보의 통합: 인터넷에 흩어져 있는 인천의 관광지, 맛집, 교통 정보를 하나의 앱에 모아 제공함으로써, 사용자가 정보 탐색에 들이는 시간을 줄이고 여행 자체에 집중할 수 있게 한다
	5. NFT 기반의 여행 기록: 여행지 방문이라는 아날로그 경험을 QR코드 스캔을 통해 고유한 디지털 자산(NFT)으로 변환시켜, 단순한 기념사진을 넘어선 새로운 방식의 소유와 기록의 가치를 부여한다
6. 서비스 미션과 비전
	- 미션: 관광객들이 인천 여행을 간편하고 즐겁게 할 수 있도록 돕는다
	- 비전: 한국을 방문한 모든 관광객들이 이용하는 플랫폼이 되는 것

## 3. 이용 프로세스

1. 시나리오
	1. 'MBTI와 체질 검사하기' 버튼을 눌러 MBTI와 체질에 대한 간단한 설문을 진행하면 사용자의 MBTI와 체질에 대한 정보를 제공받는다
		- 기능 목록
			- 메인 화면에서 `MBTI와 체질 검사하기 버튼` 표시 기능
			- MBTI와 사상의학을 설명하는 화면 표시 기능
			- 검사화면으로 이동하는 기능
			- 선택지 클릭 기능
			- 선택을 모두 완료한 사용자에게 개인의 MBTI와 체질을 설명해주는 화면 기능
			- 설명 화면에 `당신의 MBTI와 체질 맞춤 여행코스 확인하기 버튼` 표시 기능
			- 해당 사용자의 MBTI와 체질 저장 기능
		- 정책
			- 사용자는 언제든지 이전 선택지로 이동할 수 있어야 한다
			- 사용자는 언제든지 메인화면으로 나갈 수 있어야 한다
			- MBTI와 체질을 DB에 저장하는 중에는 '앱 종료 하지 마세요'메시지가 나오며 앱을 종료하면 데이터 저장을 실패한다
	2. 생성된 정보를 바탕으로 '맞춤형 코스' 버튼을 누르면 생성형 AI가 MBTI와 체질에 맞는 코스를 제시해준다
		- 기능 목록
			- 사용자가 검사를 하지 않았으면 검사화면으로 이동하는 버튼 표시 기능
			- 사용자 데이터에 맞는 여행 코스를 추천해주는 기능
		- 정책
			- 사용자 테이블에 여행 코스 데이터를 저장해야 한다
	3. 추천해준 코스에 있는 여행지에 있는 'QR을 찍으면' NFT를 발행한다
		- 기능 목록
			- 앱 내부에서 QR 코드를 찍는 기능
			- NFT를 발행하는 기능
		- 정책
			- 계정당 1회 원칙: 하나의 여행지 NFT는 한 계정당 한 번만 발행 가능하게 한다
			- GPS 인증 병행: QR 스캔 시 앱은 사용자의 GPS 정보를 함께 확인하여, 현장에 실제로 방문한 경우에만 NFT가 발행되도록 한다. 
	4. `나의 NFT갤러리`에 들어가면 획득한 NFT를 모두 보여준다.
		- 기능 목록
			- 현재까지 사용자가 발행한 NFT를 모두 모아서 보여주는 기능
		- 정책
			- NFT가 투기 목적이 아닌 디지털 기념품이자 혜택을 위한 '증표'임을 명확히 고지한다. 
	5. 추가 가능한 시나리오
		1. '지도' 버튼을 누르면 인천 전체 관광 데이터를 보여준다
		2. 지도에서 '검색'을 하면 해당 키워드에 맞는 정보만 보여준다

2. 사용자 플로우 차트
	1. MBTI와 체질 검사
		-  `홈 화면` → 사용자가 `MBTI와 체질 확인하기` 터치 → MBTI 검사 화면 → 사용자가 선택하여 터치 x N번→ MBTI 검사 종료 → 사용자가 선택하여 터치 x N번 → 사상의학 검사 종료 → '처리중' 메시지 → MBTI 및 체질 정보 제공
			- 분기
				1. (선택하지 않고 `다음` 터치) → 해당 화면에서 '선택 요구'메시지 출력(화면 유지)
				2. (선택하고 `다음` 터치) → 다음 선택 화면으로 이동(화면 이동)

	2. 맞춤 코스 
		-  `검사 완료 화면`이나 `홈 화면`에서 `맞춤 코스 확인하기` 터치 → 미리 생성해둔 코스와 사용자를 사상하여 코스 제공
			- 분기
				1. (검사를 하지 않고 `맞춤 코스 확인하기` 터치) → 'MBTI및 체질 검사 필요' 화면 출력 + `MBTI와 체질 확인하기` 버튼 출력(화면 이동) → 버튼 클릭하면 검사 페이지로 이동(화면 이동)
				2. (검사를 완료하고 `맞춤 코스 확인하기` 터치) → 사용자에게 맞는 코스를 화면에 보여줌
	3. NFT 발행
		-  `홈화면`에서 `QR코드 찍기` 버튼 터치 → 사용자가 QR코드 인식 → 인식한 QR코드에 맞는 NFT 발급
			- 분기
				1. (QR코드 인식 실패) → 'QR코드 인식 실패' 메시지 팝업 → 사용자가 `다시 촬영` 터치 → QR코드 재 촬영
				2. (QR코드 인식 성공) → NFT 발행 처리 화면으로 이동 → NFT 발급 완료
					1. (NFT 발급 실패) → 'NFT 발급 실패' 메시지 팝업 → 사용자가 `QR코드 다시 촬영` 터치 → NFT 발행 처리 화면으로 이동 → NFT 발급 완료
					2. (NFT 발급 성공) → 'NFT 발급 완료' 메시지 팝업 → 사용자가 `확인` 터치 → 발급 완료한 NFT 보여주는 화면으로 이동
	4. NFT 갤러리
		- `홈화면`에서 `NFT 갤러리` 버튼 터치 → `NFT 갤러리`로 이동
			- 분기
				- (NFT 갤러리 로딩 실패) → 'NFT 갤러리 로딩 실패' 메시지 팝업(화면 유지) → 사용자가 `확인` 터치 
				- (NFT 갤러리 로딩 성공) → `NFT 갤러리` 화면으로 이동
