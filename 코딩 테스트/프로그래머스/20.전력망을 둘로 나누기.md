## 1. 문제
n개의 송전탑이 전선을 통해 하나의 [트리](https://en.wikipedia.org/wiki/Tree_\(data_structure\)) 형태로 연결되어 있습니다. 당신은 이 전선들 중 하나를 끊어서 현재의 전력망 네트워크를 2개로 분할하려고 합니다. 이때, 두 전력망이 갖게 되는 송전탑의 개수를 최대한 비슷하게 맞추고자 합니다.

송전탑의 개수 n, 그리고 전선 정보 wires가 매개변수로 주어집니다. 전선들 중 하나를 끊어서 송전탑 개수가 가능한 비슷하도록 두 전력망으로 나누었을 때, 두 전력망이 가지고 있는 송전탑 개수의 차이(절대값)를 return 하도록 solution 함수를 완성해주세요.

## 2. 문제 이해
- 한개를 두개로 나누었을때 개수 가장 비슷하게 만들기
- 배열의 길이는 연결한 선의 개수
- 문제처럼 하나의 선을 끊으면 배열의 길이가 1 줄어듦

## 3.입력 크기 및 시간/메모리 제한 확인
- n은 2 이상 100 이하인 자연수입니다.
- wires는 길이가 `n-1`인 정수형 2차원 배열입니다.
    - wires의 각 원소는 [v1, v2] 2개의 자연수로 이루어져 있으며, 이는 전력망의 v1번 송전탑과 v2번 송전탑이 전선으로 연결되어 있다는 것을 의미합니다.
    - 1 ≤ v1 < v2 ≤ n 입니다.
    - 전력망 네트워크가 하나의 트리 형태가 아닌 경우는 입력으로 주어지지 않습니다.

## 4. 최대 시간복잡도 확인
- $n \approx 100$ => $O(N!)$ 불가
- $10^2$,$1억=10^8$
- 따라서 $O(N^4)$까지 가능

## 8. 코드 검증
```java
import java.util.*;
class Solution {
    List<Integer>[] bucket;
    
    public int solution(int n, int[][] wires) {
        int minDiff = 9999;
        bucket = new ArrayList[n+1];
        
        for(int i = 0; i < n + 1; i++) bucket[i] = new ArrayList<>();
        for(int[] temp: wires){
            bucket[temp[0]].add(temp[1]);
            bucket[temp[1]].add(temp[0]);
        }
        
        for(int i = 0; i < wires.length; i++){
            int node1 = wires[i][0];
            int node2 = wires[i][1];
            
            boolean isVisited[] = new boolean[n+1];
            int count1 = dfs(node1,node2,isVisited);
            int count2 = n - count1;
            
            minDiff = Math.min(minDiff,Math.abs(count1-count2));
        }
        
        return minDiff;
    }
    
    private int dfs(int node1,int node2, boolean isVisited[]){
        isVisited[node1] = true;
        int count = 1;
        
        for(int i: bucket[node1]){
            if(!isVisited[i] && i != node2){
                count += dfs(i,node2,isVisited);
            }
        }
        return count;
        
    }
    
    
}
```
