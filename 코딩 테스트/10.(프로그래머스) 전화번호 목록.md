## 1. 문제 이해
전화번호부에 적힌 전화번호 중, 한 번호가 다른 번호의 접두어인 경우가 있는지 확인하려 합니다.  
전화번호가 다음과 같을 경우, 구조대 전화번호는 영석이의 전화번호의 접두사입니다.

- 구조대 : 119
- 박준영 : 97 674 223
- 지영석 : 11 9552 4421

전화번호부에 적힌 전화번호를 담은 배열 phone_book 이 solution 함수의 매개변수로 주어질 때, 어떤 번호가 다른 번호의 접두어인 경우가 있으면 false를 그렇지 않으면 true를 return 하도록 solution 함수를 작성해주세요.

- 시작 번호가 같은게 하나라도 있으면 flase
- 시작 번호가 같은게 없으면 true

## 2.입력 크기 및 시간/메모리 제한 확인
- phone_book의 길이는 1 이상 1,000,000 이하입니다.
    - 각 전화번호의 길이는 1 이상 20 이하입니다.
    - 같은 전화번호가 중복해서 들어있지 않습니다.

- N => 백만
## 3. 자료구조 및 알고리즘 
- 순회해서 값 찾기
	- 배열->완전 탐색 => $O(N^2)$
	- 해쉬: 길이=N
		- 1개 확인 $O(1)$ 
		- 해쉬에 넣기: O(1)
		- 전체 확인: $O(N)$
		- 해쉬는 "123"을 한번에 "123"이런식으로 검사하는게 아니라 1,12,123이렇게 검사하기 때문에 각 전화번호의 길이의 최대인 L(20)이 추가됨
		- 따라서 시간복잡도는 $O(N\times L)$이다
	- 트리
		- 이론상 가장 적합, 직접 구현해야 하기 때문에 해시 선호

## 4. 로직 설계
- 접두어가 "1234"라고 가정
	- "1234"가 나오면 true
1. HashMap에 넣음
2. 첫번째부터 마지막 값 까지 for
	1. 해당 인덱스가 접두어인 값이 있는지 확인
	2. 있으면 바로 false return
	3. 없으면 true

## 5. 복잡도 검증
- $O(N \times L)$

## 6. 라이브러리 선택
- set,HashSet

## 7. 검증
```java
import java.util.*;
class Solution {
    public boolean solution(String[] phone_book) {
        Set<String> bucket = new HashSet<String>(Arrays.asList(phone_book));
        
        for(String val:phone_book){
            for(int i = 1; i < val.length();i++){
                String subVal = val.substring(0,i);
                if(bucket.contains(subVal)){
                    return false;
                }
            }
        }
        return true;
    }
}
```
